# 🌟 Saigon Happy Tour - Personalized Booking & Dashboard

Hệ thống cá nhân hóa trải nghiệm đặt tour du lịch và trang quản trị doanh nghiệp (Dashboard) dành cho **Saigon Happy Tour**.

## 🚀 Tính năng chính
- **Khách hàng:** Tự động đề xuất tour phù hợp thông qua Form thu thập thông tin và AI Chatbot.
- **Doanh nghiệp:** Theo dõi Realtime Comment, Sentiment Analysis, Market Share và Tour Performance.

## 💻 Công nghệ sử dụng
- ReactJS + Vite
- Tailwind CSS (Chuẩn nhận diện thương hiệu Vàng - Xanh)
- Recharts (Vẽ biểu đồ tương tác)
- Tích hợp n8n AI Workflow
